Como funciona a Web e o que é uma página web?

Para compreender bem como a Web funciona, é legal entender sobre o conceito de Redes. Basicamente, temos um cliente e um servidor que ao se comunicarem entre si, criam o que conhecemos como internet. As páginas web formam um site (na internet) e esse site é hospedado em um servidor. Quando o cliente solicita alguma informação em uma página web, o servidor retorna as informações solicitadas. A partir dessa relação, vários computadores se comunicam no mundo todo com capacidade de hospedar páginas e sites para que nós usuários possamos acessar.

Os papéis das linguagens HTML e CSS numa página web

O HTML é uma linguagem de marcação, ele cria o "esqueleto" de uma página web, ou seja, ele define como será a estrutura daquela página e se adequa a todos os navegadores disponíveis. Por ele ser um documento de extensão .html, o navegador consegue ler esse arquivo e renderizar seu conteúdo para o usuário que o acessa. Já no caso do CSS, estamos falando de uma linguagem de estilização, criando "beleza" na pagina e alterando seus elementos estéticos, como posicionamento dos objetos, cor de fonte, tamanho de imagem, etc. Uma analogia boa para explicar os dois é pensar no HTML como uma casa recem construída, e o CSS como a decoração dessa casa.

O que são ambientes de programação?

Os ambientes de programação ou ambientes de desenvolvimento são softwares ou websites que constroem uma aplicação que um desenvolvedor codifica, ou seja, é o principal local de trabalho do desenvolvedor quando ele está codificando alguma aplicação.
